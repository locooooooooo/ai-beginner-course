# AI 新手课文本源

这个目录把课程里需要反复打磨的文字内容从页面生成结果中拆出来。

编辑方式：

1. 改 `chapters/*.toml`：章节标题、导学说明、章节产物。
2. 改 `lessons/*.toml`：每课标题、目标、问题、案例、方法、误区、讲解脚本和示例。
3. 改 `case-playbooks.toml`：所有课程共用的案例拆解文案。
4. 改完后在课程根目录运行 `py -3 scripts/generate_lessons.py` 重新生成页面。

注意：

- `slug`、`id`、图片路径和课程编号会影响链接与资源引用，除非明确要改结构，否则不要改。
- 普通文字可以直接改三引号 `"""..."""` 里的内容。
- 列表项可以增删，但要保留每行末尾的逗号。
