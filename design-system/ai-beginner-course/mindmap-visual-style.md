# AI 入门课脑图视觉风格

## 风格定位

课程脑图统一采用“复古像素贴纸 + 街头学习卡带”风格。目标不是做装饰图，而是让零基础学员在手机和网页放大状态下快速看清层级、重点和学习顺序。

## 视觉原则

- 先读主干，再读细节：中心标题、一级分支、叶子节点必须形成清晰的三级视觉层级。
- 高对比优先：正文以白底黑字为主，重点用柠檬黄和高饱和橙色标记。
- 游戏化但不幼稚：使用像素边框、硬阴影、卡带节点和小图标，但不使用复杂插画或低对比渐变。
- 统一连接语言：所有分支线使用黑色直角折线，避免彩色曲线造成噪声。
- 可维护优先：所有 PNG 从 `.mmd` 源文件批量渲染，避免手工拖拽造成风格漂移。

## 设计令牌

- 背景：`#F7F7F5`
- 墨色：`#121212`
- 主橙：`#FF5C00`
- 柠檬黄：`#FFD600`
- 白色卡片：`#FFFFFF`
- 灰色实体阴影：`#CFCFC7`
- 中心节点边框：4px 黑色像素边框
- 一级节点边框：4px 黑色像素边框
- 叶子节点边框：2px 黑色实线
- 硬阴影：6-8px 黑色或灰色实体阴影

## 节点规范

- 中心节点：柠檬黄贴纸，粗黑边，右下硬阴影，标题居中。
- 一级节点：游戏卡带造型；按列使用橙、黄、黑、白循环区分。
- 二级重点节点：白色贴纸卡，左侧 8px 彩色强调条。
- 末级知识点：白色小贴纸卡，黑色细边，轻灰硬阴影。
- 图标点缀：只用于重点类别，如 `星标 / 警示 / 盾牌 / 工具 / 检查`，避免每个节点都堆图标。

## 连接线规范

- 使用 4-5px 黑色直角折线。
- 中心节点先连到一条总线，再下接一级节点。
- 一级节点下方使用垂直主干线连接叶子节点。
- 连接节点处使用小黄方块作为像素锚点。

## 文案排版规范

- 每个叶子节点尽量控制在 1-2 行。
- 英文模型名不从单词中间断行。
- 长脑图允许横向加宽或纵向加高，不压缩字号。
- 标题不使用负字距，不随视口缩放。

## 生成方式

使用 `AI 新手 20 节课/scripts/render_pixel_mindmaps.py` 批量渲染源 `.mmd` 文件，再运行 `AI 新手 20 节课/scripts/generate_lessons.py` 同步到 `site/` 和 `github-pages-publish/`。
