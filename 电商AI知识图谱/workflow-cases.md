# 电商 AI 工作流实例

## 案例 1：自动化发布文章

目标：每天根据新品、活动或关键词生成一篇可发布文章。

输入：

- 商品名称、卖点、规格、价格、库存。
- 目标关键词、目标人群、品牌语气。
- 活动信息、优惠券、落地页链接。

Agent 流程：

1. 从商品库读取商品资料。
2. 从关键词表或搜索数据选择选题。
3. 用 `SEO文章 Skill` 生成标题、大纲和正文。
4. 用图片工具生成封面图或从素材库选图。
5. 做合规检查：价格、功效、夸大承诺、违禁词。
6. 创建 CMS 草稿。
7. 人工审批后发布。
8. 自动生成社媒摘要并排程。
9. 第二天读取访问、点击和转化数据，生成复盘。

需要的 Skill：

- SEO 文章结构模板。
- 标题生成与筛选规则。
- 品牌语气规范。
- 合规与禁用词清单。
- 发布前检查清单。

可能用到的 MCP/工具：

- 商品库或 Shopify。
- WordPress、Shopify Blog 或自建 CMS。
- 图片库或 Canva。
- Buffer 等社媒排程工具。
- Google Analytics、Search Console 或站内统计。

验收：

- 文章 URL 可访问。
- 标题、封面、正文、CTA、商品链接齐全。
- 没有价格错误、库存错误、夸大承诺。
- UTM 和数据回流正常。

## 案例 2：自动化生成图片

目标：针对一个 SKU 批量生成主图、详情图、广告图和朋友圈海报。

输入：

- 商品白底图或实拍图。
- 品牌色、字体、Logo、版式规范。
- 平台尺寸：主图、详情页、Instagram、Facebook、朋友圈、小红书。
- 活动信息和利益点。

Agent 流程：

1. 读取商品资料和卖点。
2. 选择图片类型：主图、场景图、广告图、海报。
3. 根据 `电商图片 Skill` 生成提示词。
4. 调用图像生成或设计工具生成多版本。
5. 检查商品是否变形、文字是否错误、价格是否一致。
6. 输出多尺寸版本。
7. 人审通过后上传素材库。
8. 自动关联商品、活动或广告计划。

需要的 Skill：

- 电商图提示词模板。
- 平台尺寸表。
- 品牌视觉规范。
- 图像质检清单。

可能用到的 MCP/工具：

- Canva、AdCreative.ai、Adobe Firefly、Midjourney 或自建图像模型。
- 商品素材库。
- 广告平台。
- CMS 或店铺后台。

验收：

- 商品主体准确。
- 字体、颜色、Logo 符合品牌规范。
- 不出现错误价格、错误规格、虚假功效。
- 文件命名和尺寸符合投放要求。

## 案例 3：自动化获客

目标：把公域互动变成可跟进的线索。

输入：

- 用户评论、私信、表单、广告点击。
- 来源渠道和活动名称。
- 用户兴趣点。

Agent 流程：

1. 监听评论、私信、表单或广告线索。
2. 判断用户意图：咨询、领券、比价、售后、无效。
3. 自动回复或发送领取入口。
4. 生成客户标签和线索评分。
5. 写入 CRM 或私域系统。
6. 分配导购或销售。
7. 根据跟进结果更新标签。

需要的 Skill：

- 首轮私信话术。
- 线索评分规则。
- 渠道归因规则。
- 低意向用户培育 SOP。

可能用到的 MCP/工具：

- Manychat。
- 广告线索表单。
- CRM。
- 短信、邮件、WhatsApp、企微。
- Zapier、Make、n8n 或 Dify。

验收：

- 线索来源可追踪。
- 回复不违规、不骚扰。
- CRM 内有标签、来源、下一步跟进任务。
- 转化数据能回流。

## 案例 4：公域转私域

目标：把平台流量沉淀为可反复触达的客户资产。

输入：

- 公域来源：短视频、直播、广告、搜索、内容种草。
- 用户动作：关注、评论、领券、下单、加购。
- 私域入口：企微、微信群、WhatsApp、邮件、短信、会员系统。

Agent 流程：

1. 判断用户来源和兴趣。
2. 给出权益诱因：优惠券、资料、试用、会员权益、售后服务。
3. 引导用户订阅、加好友或入群。
4. 进入欢迎 SOP。
5. 根据行为打标签。
6. 推送内容、活动、导购任务。
7. 追踪首单、复购和流失风险。

需要的 Skill：

- 欢迎语模板。
- 三天破冰 SOP。
- 七天转化 SOP。
- 老客复购 SOP。
- 私域群发频率规则。

可能用到的 MCP/工具：

- 企微 SCRM。
- WhatsApp Business。
- 邮件短信平台。
- 会员系统。
- 优惠券系统。
- 有赞、尘锋、微伴等私域工具。

验收：

- 用户授权明确。
- 标签准确。
- 群发频率合规。
- 导购跟进有记录。
- 私域转化和复购可统计。

## 案例 5：AI 客服与导购

目标：让 AI 自动处理高频问题，同时把复杂问题交给人工。

输入：

- 客户问题。
- 商品、订单、物流、库存、退换货政策。
- 客户等级和历史购买。

Agent 流程：

1. 识别意图。
2. 查询订单、库存、物流或政策。
3. 生成符合品牌语气的回复。
4. 必要时执行动作：发优惠券、创建退货请求、推荐商品。
5. 高风险问题转人工。
6. 对每次对话做质检和复盘。

需要的 Skill：

- FAQ 回复模板。
- 售后政策解释模板。
- 导购推荐规则。
- 人工接管规则。

可能用到的 MCP/工具：

- Gorgias、Klaviyo Customer Agent。
- Shopify、ERP、物流系统。
- CRM、优惠券系统。

验收：

- 回复准确。
- 订单和隐私信息不泄露。
- 高风险问题能转人工。
- 有客服质检记录。

## 案例 6：复购与会员运营

目标：让老客自动进入不同生命周期运营流程。

输入：

- 购买频次、客单价、最近购买时间。
- 商品消耗周期。
- 浏览、加购、点击、退订行为。

Agent 流程：

1. 做 RFM 或生命周期分层。
2. 预测下一次购买窗口。
3. 生成对应文案和优惠策略。
4. 通过邮件、短信、私域、App Push 触达。
5. 跟踪打开、点击、下单。
6. 根据结果自动优化内容和频率。

需要的 Skill：

- 生命周期分层规则。
- 优惠策略边界。
- 会员权益模板。
- 流失召回 SOP。

可能用到的 MCP/工具：

- Klaviyo、Omnisend。
- CRM、会员系统。
- 优惠券、订单系统。

验收：

- 退订机制可用。
- 优惠不伤毛利。
- 复购数据可归因。
- 高频打扰被限制。
